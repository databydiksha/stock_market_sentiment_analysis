**Stock Market Movement Prediction Using Sentiment Analysis**

Overview:
This project predicts stock market movement using sentiment analysis and Machine Learning techniques.

Features:
- Sentiment Analysis
- Exploratory Data Analysis (EDA)
- Logistic Regression Model
- Market Movement Prediction

Technologies Used:
- Python
- Pandas
- Matplotlib
- Scikit-learn

Project Outcome:
The model predicts whether the stock market is likely to move up or down based on sentiment scores.
